# 5-Minute Training Video Script

## 0:00–0:30 — Intro
"Welcome to GOD'S EYE. In this walkthrough we'll log in, enter a sale, review dashboards, use AI insights, and finalize budget controls."

## 0:30–1:10 — Login + Navigation
- Show login form.
- Sign in as management.
- Highlight sidebar groups: operational pages, dashboards, admin tools.

## 1:10–2:00 — Enter Sale Here
- Open Enter Sale Here.
- Fill all required fields.
- Click ENTER SALE.
- Call out success toast and automatic dashboard refresh.

## 2:00–2:45 — Sales Detail Page
- Open Sales Detail Page.
- Edit COST and SOLD FOR on one row.
- Toggle CASHED and point out BY timestamp initials format.
- Mention role controls: associates cannot edit after submit.

## 2:45–3:45 — Dashboard Tour
- SALES PERFORMANCE: month selector, pacing metrics, over/under colors.
- Budget 2026: editable cells and finalization lock.
- OVERALL SALES + one category dashboard (IN PERSON vs REMOTE).
- Mention all additional dashboards are formula-matched to Excel.

## 3:45–4:30 — AI & Recommendations
- Ask AI: "Who is under budget this month?"
- Ask AI: "Forecast Noah's GP for the rest of the year."
- Show anomaly response and smart recommendations panel.
- Click Generate Executive Summary PDF.

## 4:30–5:00 — Admin + Mobile/PWA
- Open Manage Employees and deactivate/reactivate sample employee.
- Open Manage Dropdowns and add one lookup value.
- Show install prompt for PWA on mobile view.
- Close: "You're ready to run GOD'S EYE end-to-end."
