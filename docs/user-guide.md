# GOD'S EYE User Guide

## 1. Login
1. Open the app URL.
2. Sign in with email/password.
3. You land on Dashboard Home with daily insights.

## 2. Enter Sale Here
1. Open **Enter Sale Here** from sidebar.
2. Fill Product ID, Brand, Reference, Type, Date In, Date Out, Customer Name, In Person, Lead Source, Sold For.
3. Click **ENTER SALE**.
4. Success toast appears and dashboards refresh automatically.

## 3. Sales Detail Page (Management/Admin)
1. Open **Sales Detail Page**.
2. Edit **COST** or **SOLD FOR** inline.
3. Toggle **CASHED** to mark payout and auto-stamp `BY` as initials + date.
4. Changes recalculate KPIs and pacing dashboards.

## 4. Dashboards
- SALES PERFORMANCE: monthly pacing by associate.
- Budget 2026: editable planning fields until finalization.
- OVERALL SALES: annual associate roll-up.
- IN PERSON vs REMOTE: category split with margin and GPPU.
- LEAD PERFORMANCE Monthly/Annual/M2M.
- INVENTORY TIERS.
- BRAND PERFORMANCE + BRAND PERF M2M.
- INVENTORY MIX + INVENTORY MIX PER SALES PERSON.

## 5. Budget Finalization
1. Go to **Budget 2026**.
2. Edit inventory budget, average inventory value, and margin budget.
3. Click **Finalize Budget**.
4. Lock icon appears and editing is disabled.

## 6. Admin Tools
### Manage Employees
- Add employee with name, email, initials, role.
- Deactivate employee to remove from active dashboard roster for future reporting.

### Manage Dropdowns
- Add/deactivate values for brands, lead sources, condition types, and other lookups.

### Condition Scale
- Read-only reference for grading standards.

## 7. AI Features
- **Ask Larry** page for AI Q&A, forecasts, anomalies.
- Smart Recommendations card on each dashboard.
- Dashboard Home: Daily insights + alerts.
- One-click **Generate Executive Summary PDF**.

## 8. Mobile & PWA
- App is responsive across phone/tablet/desktop.
- Install prompt appears when supported.
- Installed app works as a PWA shell with offline cache fallback.

## 9. Troubleshooting
- If AI responses fail, verify `XAI_GROK_API_KEY`.
- If data is missing, verify user role and active status.
- If dashboards look stale, refresh browser; realtime sync should reconnect automatically.
